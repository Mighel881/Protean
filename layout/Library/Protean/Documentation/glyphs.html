How Does One Create Icons for Protean?

Glyphs/icons can easily be created for Protean. If youâ€™re planning to create an icon or two, follow these instructions:

Instructions
  1. Launch your favourite graphic design application.
  2. Create a canvas with a 1:1 aspect ratio and a resolution of 20x20 or higher.
  3. Design the icon.
  4. Save the image as a PNG with a resolution of 20x20.
  5. Name the image "PR_Desired Name@2x" (without the quotation marks).
  6. Transfer the icon to /Library/Protean/Images.bundle on your iPhone.
  7. ???
  8. Profit!
General Notes
  • The width of the icon can be above 20px's.
  • If you are planning to make count icons, make sure to follow this filename format:
    • PR_Desired Name_Count_1@2x.png, PR_Desired Name_Count_2@2x.png, PR_Desired Name_Count_3@2x.png, and so on...
  • If you'd like a count icon for anything above what you have done, name it PR_Desired Name_Count_Large@2x.png.
