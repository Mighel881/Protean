Introduction

The Advanced Options pane includes additonal Protean features that allow for further customization of your status bar.

Item Spacing

Allows you to change the amount of space between status bar items

Number of Spacers

Spacers are items that can be placed between status bar items.


Battery Percentage Styling

Protean gives you the option of eight different options when it comes to your battery percentage. Details about the options are listed below. Please note, the actual battery percentage options may not be function properly on iOS devices that do not have the hardware support reading the battery's charge, capacity, etc.

  • Default: The battery percentage iOS displays by default, calculated from Apple's algorithm. The algorithm uses a running average of your battery usage and incorporates this into the current percentage. This results in the percentage showing your "remaining battery" rather than your current charge.

  • Hide % sign: The default battery percentage with no percentage sign.

  • Textual: The default battery percentage written out.

  • mAh charge: The current charge value of your battery

  • Actual percentage: Calculated by taking your battery's charge (mAh) and dividing it by the battery's current max capacity

  • Longer actual percentage: The actual battey percentage to the hundredth of a percent

  • Longer actual percentage with no %: The actual battey percentage to the hundredth of a percent, with no percentage sign.

  • Textual with percent: The default battery percentage written out, with 'percent' at the end.

Battery Percentage Color

This section allows you to assign a custom color, using Red Green Blue and/or Hex values, to the battery percentage.
