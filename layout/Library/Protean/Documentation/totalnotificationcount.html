Total Notification Count
Introduction

This section of Protean allows you to add a glpyh to the status bar that displays the total number of notifications you currently have.

Note: In order for this feature to function as a intended you must select a glyph that supports badging/numbering. The stock Protean glyphs all support this feature.

Tap Actions

Once you have selected your flipswitch, you will see the following screen. On this screen you will are able to select the glyph you want to use for the application as well as the application's "Tap Action."

Tap Actions occur when you tap the application's glyph in the status bar. Selecting "Launch Application" will cause the application to launch when you tap the glyph. If you select "Activator Action" you will be taken to a list of activator actions that you can select.
